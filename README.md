# java-university

Консольна система керування університетом на Java.

## Що вміє програма

- CRUD для студентів, викладачів і курсів.
- Зарахування студентів на курси.
- Виставлення оцінок і позначення оплати.
- Підрахунок GPA студента.
- Пошук студентів за ПІБ або email.
- Звіти по неоплачених курсах, середньому GPA та топ-N студентів.
- Фільтри студентів за статусом/роком і курсів за викладачем/кредитами.

## Структура

```text
src/university/
  Main.java
  entities/
  enums/
  interfaces/
  services/
  util/
```

## Запуск

```bash
javac -d out $(find src -name "*.java")
java -cp out university.Main
```

Після запуску в консолі відкриється меню. У програмі вже є кілька тестових студентів, викладачів, курсів і зарахувань, щоб швидко перевірити роботу.
